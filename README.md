# Truna Data Helper

![npm](https://img.shields.io/npm/v/npm)

Web scraper for mansionbet and nfl games schedule with simple UI witch visualise data from espn api and mansionbet.

## Built With

- React
- Node.js
- Express
- Puppeteer
